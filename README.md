# Second Hand Car Price Analysis using Machine Learning

## Project Overview
This project analyses second-hand car sales data using both supervised and unsupervised machine learning models.
In this exercise, It is tried to analyse a mock dataset of second hand car sales in the UK to
find out how to predict car prices accurately using different supervised learning models.
It has also been tried to cluster the data set as much as possible to find a good pattern in
the sold cars using unsupervised learning models.

## Objectives
- Explore the dataset using EDA
- Predict car prices using regression models
- Discover patterns using clustering techniques

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Models

Supervised:
- Linear Regression
- Random Forest

Unsupervised:
- KMeans Clustering


## Results
The comparative analysis in the metrics table and graphs clearly identified the random
forest model as the most effective tool for price prediction with a slight difference
compared to the hypertuned neural network model. Both of these models can predict car
prices accurately and their superior performance were clearly evident in Table 5 showing
the highest R2 value and the lowest MSE among all evaluated models.
